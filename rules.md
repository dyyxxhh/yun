# 游戏规则文档

## 道具列表

### 1. Magic 运
- **防御**: 0
- **攻击**: 0
- **效果**: +3魔力

### 2. Magic 气波
- **防御**: 0
- **攻击**: (魔力 + 运) ÷ 3

### 3. Magic 防
- **防御**: 4
- **攻击**: 0
- **消耗**: 2点魔力

### 4. 土
- **防御**: 3
- **攻击**: 0
- **效果**: 增加1次死亡机会（优先级比杀死低）
- **生效时间**: 从出道具的第2回合开始（提前结算下回合消耗的）
- **消耗**: 每回合消耗2点魔力
- **限制**: 每局最多出1次，断了失去效果

### 5. 刺果
- **防御**: 0
- **攻击**: 0
- **消耗**: 1运
- **条件**: 需要魔力 ≥ 3
- **效果**: 对方下回合防御 = min(攻击, 防御)
- **备注**: 此道具为单体道具

### 6. 阉割
- **消耗**: 1运
- **条件**: 如果对方出的是运或者 Magic 运
- **效果**: 
  - 对方再也无法获得运或者魔力
  - 防御 = 1
  - 攻击 = 0
- **备注**: 此道具为单体道具

### 7. Magic 星尘
- **生效时间**: 从用过的第2回合开始
- **效果**: 
  - 被选中的玩家每回合增加1点辐
  - 可用"结算"结算伤害
  - 每名玩家最多结算1次或被结算1次（这两个单独计算）
  - 伤害由中立造成，无视准心腐化，没有亡语
- **攻击**: 0
- **防御**: 0
- **结算效果**: 攻击 = 防御 = 0

### 8. Magic 枪

- **效果**:

  - 其右方人受到(X - 选中人防御)的伤害，以此类推
  - 如果所有人死，则使用该道具的人赢

- **备注**: 此道具为单体道具（虽然算单体道具，但是如果复制失败视为0攻0防，如果复制成功那么就不是单体）

### 17. 破解复制
- **攻击**: 0

### 18. 强制

- **惩罚**: 出不出来，对方则自爆
- **限制**: 强制之后的那1回合不能出玩手机或者魔法玩手机（例如炸弹等物品）
- **备注**: 不同状态视为同1物品
- **备注**: 此道具为单体道具

### 19. 防
- **防御**: 3
- **攻击**: 0
- **消耗**: -0运

### 20. 大防
- **防御**: 17
- **攻击**: 0
- **消耗**: -1运

### 21. 炸弹
- **消耗**: -2运
- **攻击序列**: 先出攻=1，再出攻=2，再出攻=3，再出攻=4
- **防御**: 始终为0
- **限制**: 总共最多断1回合，否则自爆
- **示例**: 
  - 可以出：1运234
  - 不能出：1运运234
  - 不能出：1运 2运 34

### 22. Magic 炸弹
- **消耗**: 从初次使用即立即扣除所需魔力，每1个消耗1.5魔力，没有上限
- **示例**: 假设你有15点魔力，那么会立即扣除所有的魔力
- **其他**: 同炸弹，但是4个之前无法中断
- **中断规则**: 4个之后中断之后需要重新计数，中断之后会返还没有使用的魔力
- **特殊情况**: 如果魔力小于1.5点，那么多出来的那部分不会被消耗，也不会被扣留
- **叠加规则**: Magic炸弹与Magic炸弹的伤害不会叠加，但是与Magic炸弹-复制版的伤害会叠加

### 23. 格挡
- **消耗**: -1运
- **防御**: 3
- **效果**: 若此回合有人攻击你，且你没死，下回合的防御加17

### 24. 玩手机
- **条件**: 若所有目标都出防或者大防（注意：不包含Magic防）
- **可选条件**: 如果可选择含队友，并且需要至少有1个敌人出防或者大防
- **效果**: 加3个运（此操作不消耗运）
- **攻击**: 0
- **防御**: 0

### 25. 准星
- **消耗**: -1运
- **效果**: 使指定人成为场上唯1可选中的目标
- **备注**: 自杀无视，但仅限于被准星的人
- **冲突规则**: 当场上在同1回合同时出现两个准星，且指向不同的目标，那么无论当前是否有人被准星，皆清除场上准星状态
- **注意**: 准星无视准星的影响
- **使用条件**: 使用该道具需要人数大于2
- **副作用**: 在使用准星期间，无论原本使用者的防御为多少，强制防御=0
- **惩罚**: 若于使用的回合受到任何伤害，则使用者立即被伤害来源杀死
- **生效时间**: 准星自下回合才开始生效
- **备注**: 此道具为单体道具

### 26. 腐化
- **条件**: 需要人大于2
- **消耗**: -1运
- **效果**: 强制被准星，但自己使用的道具不受到准星的影响

#### 腐化规则详解：
1. **腐化覆盖**: 当场上存在腐化者，且有非腐化者出腐化并成功结算时，腐化者失去效果，且出腐化的那个人不腐化
2. **伤害计算（非准星后腐化）**: 当场人存在腐化者，且腐化者不是在被准星之后腐化的，非腐化者对腐化者的伤害 = 原伤害 × 假设无腐化的目标数
3. **伤害计算（准星后腐化）**: 如果腐化者是在被准星之后腐化的，非腐化者对腐化者的伤害 = 除去被腐化者所有人原伤害之和
4. **腐化者伤害**: 腐化者的对其他人伤害乘以2

### 27. 右手螺旋定则
- **消耗**: -2运
- **效果**: 将自己受到的伤害转移给右边的人（跳过友军）
- **防御**: 3
- **备注**: 此道具为单体道具

### 28. 聚变
- **效果**: 指定两个人，分别是出方跟入方
- **伤害计算**: 出方将受到入方造成的（原目标数 × 入方伤害）的伤害
- **限制**: 入方不对其他人造成伤害
- **防御**: 3
- **消耗**: -2运

### 29. YGG

### 39. 钉子

### 40. 锤子

- **钉子联动**: 若使用对象已经有至少1颗钉子，那么对方直接加1颗钉子
- **备注**: 此道具为单体道具

### 41. Magic YGG
- **目标**: 对1名玩家（可为活人，可为自己）使用
- **消耗**: -0运，-0魔力
- **防御**: 3
- **攻击**: 0
- **效果**: 从使用的下1回合开始，每回合失去0.5点魔力，持续6个回合，并于第6回合复活该玩家
- **惩罚**: 若在第6回合该玩家活着（当回合死亡不算活着），则你自爆
- **死亡处理**: 若使用者死亡，可正常扣除魔力并生效
- **限制**: 被Magic YGG的人无法被YGG或者被Mark，直到效果结束
- **失败惩罚**: 若因魔力不足导致失败，则该玩家本局游戏结束，无法使用Magic和YGG
- **交互规则**: 若1个人同时被Magic YGG跟YGG，则视为YGG的效果失效（保留1点防御）
- **备注**: 此道具为单体道具
- **魔力扣除**: Magic YGG扣除魔力与判定是否保留状态机制与土大致1致，魔力在结束阶段扣除

### 42. 邪眼护符
- **消耗**: -2运
- **防御**: 2
- **触发条件**: 若在出该道具的回合没有受到忽视防御伤害

- **效果**: 则下3个回合内的防御加2
- **失效条件**: 受到忽视防御伤害则失效（保留初始出的那1回合的2点防御）

## 伤害计算

**前提**: A、B为队友

### 情况1
- A出上勾，B出上勾，C出防（3点防御）
- 此时A对C造成2点伤害，B对C造成2点伤害
- C受4点伤害，则C死亡

### 情况2
- A出上勾，B出上勾，C出上勾
- A对C造成两点伤害，B对C造成两点伤害
- C对A造成两点伤害，C对B造成两点伤害
- 伤害均抵消，所以C不死

## 补充规则

### 1. 目标无效规则
- 当单体道具指定到无法指定的玩家时，该道具不生效，但正常消耗

### 2. 恐惧震慑
- **定义**: 当玩家A尝试复制玩家B的道具，且玩家B对玩家A的伤害大于5，则该玩家立即复制失败
- **效果**: 被恐惧震慑算复制失败

### 3. 无法使用道具规则
- 当1玩家因规则在某1回合无法使用道具，则视为0攻0防

### 4. Magic 放叠加规则
- 多名玩家同时对1名玩家使用Magic放时，伤害正常造成，只是不给魔力（mana是魔力的意思）

### 5. 自爆规则
- **定义**: 因用错道具（即规则不让出）或者违反规则，则该玩家立即被抹除

### 6. 杀死抵消规则
- A玩家对B玩家的杀死（或抹除）可以抵消B玩家对A玩家的所有伤害

### 7. 杀死与抹除定义
- **杀死**: 无论被杀死的玩家以何种姿态，该玩家直接正常死亡（无视土）
- **抹除**: 该玩家直接正常死亡，随后被移出游戏（无法被YGG等）

### 8. 平局判定规则
- 当场上所有存活玩家在同1回合死亡（或被抹除），因此无法判定胜负时，按下列顺序判定（大者为胜）：
  1. 受N点伤害 > 受N+1点伤害 > 被杀死 = 被抹除 > 被杀死加受N点伤害 = 被抹除受N点伤害 > 被杀死受N+1点伤害 = 被抹除受N+1点伤害

### 9. 团战规则
- 没有友军伤害
- 禁用普枪等道具

### 10. 锤子结算顺序
- 当某1道具被锤子砸之后，先结算用锤子者是否死亡，再结算锤子照相机照复制为复制到的道具

### 11. 循环复制规则
- 循环复制（A复制B，B复制A）视为0攻0防

### 12. YGG自用规则
- YGG等道具可以给自己用，但是如果用的当回合自己没死，则自己自爆（因为给活人用了YGG）
- 如果当时死了也救不起来，因为YGG没有死亡后的效果
- 被复活的人从被复活的下1回合开始才能出道具（当回合不算存活数）
